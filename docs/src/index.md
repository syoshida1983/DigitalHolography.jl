!!! note
    When the object wave is denoted as ``u_{o} = ae^{i\phi}`` and the reference wave as ``u_{r} = be^{i\delta}``, the interference fringe ``I`` can be represented as ``I = |u_{o} + u_{r}|^{2} = a^{2} + b^{2} + 2ab\cos(\phi - \delta)``.
